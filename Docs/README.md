## 主题

默认主题为 `sphinx_ustack_theme` ，如果计算机没有安装，请执行以下命令安装主题：

```
pip install sphinx_ustack_theme
```

## 文档生成

项目文档采用 `Sphinx` 生成，`reStructuredText` 书写。

* For Windows. PowerShell

```bash
PS> cd OSDA/Docs
PS> .\make.bat html
```

* For Linux

```bash
$ makefile html
```
