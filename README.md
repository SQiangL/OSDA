### 介绍
---------------------
这是一个开放源代码，以TIM授权的串行端口调试助手。

### 软件架构
---------------------
采用WPF框架，MVVM模型。

### 功能
---------------------
- [X] 建立完整的MVVM模型
- [X] 基础功能（打开、关闭、常规接收、常规发送和清空）
- [ ] 十六进制接收，十六进制发送
- [ ] 多项发送
- [ ] 自动发送（单位毫秒）
- [ ] 接收保存及路径选择（接收自动保存）
- [ ] 在线更新
